# 💱 Conversor de Moedas

Converta valores entre **Real (BRL)**, **Dólar (USD)**, **Euro (EUR)** e **Libra Esterlina (GBP)** de forma rápida, prática e intuitiva.

**HTML • CSS • JavaScript**

---

## 📌 Sobre o projeto

O **Conversor de Moedas** é uma aplicação web desenvolvida com **HTML5, CSS3 e JavaScript**, criada para facilitar a conversão entre diferentes moedas utilizando uma interface moderna, responsiva e intuitiva.

O usuário escolhe a moeda de origem, a moeda de destino, informa o valor desejado e obtém instantaneamente o resultado da conversão.

O projeto foi desenvolvido para praticar manipulação do DOM, eventos, funções, formatação monetária e organização de código. Também está preparado para futuras integrações com APIs de câmbio em tempo real.

---

## ✨ Funcionalidades

* 💱 Conversão entre Real (BRL), Dólar (USD), Euro (EUR) e Libra Esterlina (GBP)

* 🔄 Fácil expansão para novas moedas

## 🚀 Como usar

Clone o repositório:

```bash
git clone https://github.com/Isaak7s/conversor-moedas.git
```

Entre na pasta do projeto:

```bash
cd conversor-moedas
```

Abra o arquivo **index.html** no navegador ou publique o projeto utilizando o GitHub Pages.
